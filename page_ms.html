Metal Soldiers

Metal Soldiers is a "Mech Customization Action" game from the creator of Kingdom Come Rising!

These are the minimum requirements to run it:

  • CPU: Any decent 64-bit processor
  • GPU: Integrated graphics work fine
  • RAM: 2GB or more, 4GB recommended
  • OS: Any 64-bit Linux distribution, 64-bit Windows (10 or newer is recommended), or any OS capable of running Browser games
