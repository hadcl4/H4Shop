Kingdom Come RUSH

Kingdom Come RUSH is an endless strategy game featuring characters from Kingdom Come Rising.

These are the minimum requirements to run it:

  • CPU: Any decent 64-bit processor
  • GPU: Integrated graphics work fine
  • RAM: 2GB or more, 4GB recommended
  • OS: Any 64-bit Linux distribution, 64-bit Windows (10 or newer is recommended), or 64-bit Android
